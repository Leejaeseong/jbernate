		- View 조회 권한 : 사용자가 속한 그룹에서 해당 View에 대한 권한이 있어야 조회되도록 설계하고, 사용자 그룹핑 및 그룹과 View를 매핑하는 화면을 개발하여 제어
	■ 오류처리

	■ 패키지 네이밍
		▷ prefix
			- com.jbernate
		▷ content
			- cm : common( 공통 )
			- tt : test( 테스트 )
		▷ postfix
			- controller 		: 컨트롤러 패키지
			- service			: 서비스 인터페이스 패키지
			- service.impl		: 서비스 구현 패키지
			- dao				: Data access object 인터페이스 패키지
			- dao.impl			: Data access object 구현 패키지
			- domain.table		: ORM 테이블 객체 패키지
			- domain.view		: ORM 뷰 객체 패키지
			- temp				: 임시 패키지( 주로 테스트 용도 )
			- testcase.crud	: crud 관련 테스트케이스 패키지
			- testcase.temp	: 임시 테스트케이스 패키지( 주로 테스트 용도 )
			- bean				: 공통 항목을 가진 빈 객체 모음 패키지
			- util				: 유틸리티 패키지
			- view				: Viewr esolver 관련 패키지
		※ 공통에서 네이밍 규칙에 의해 처리를 하므로 규칙 위반 시 뷰를 읽어오지 못하는 현상 발생
		
	■ 실제 테이블 스키마를 읽기 위해 Hibernate에서 인식하는 네이밍 룰( Alias가 없을 경우 )
		- ex) 실 테이블 스키마 : TT_ONE_TABLE_V => Hibernate : TtOneTableV
